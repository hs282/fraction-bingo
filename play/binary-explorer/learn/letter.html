Writing Letters in Binary

Previous

Now we know how binary is used to represent numbers. Next, we will discover how to create different letters using binary.

The way we use binary to make letters is to represent each letter in the alphabet by a number. The standard way of representing letters with numbers is called ASCII (American Standard Code). ASCII uses 7 digits to represent all the letters in the alphabet and some special characters.

For example, in ASCII we can say "A" is 65 and "Z" is 90. So, "A" is 1000001 and "Z" is 1011010.

For lowercase letters, "a" is 97 (1100001) and "z" is 122 (1111010).

Welcome to Binary Explorer!

Here, we will learn about binary, the language of computers. In binary, there are only two digits, 1's and 0's. Since only two digits are possible, we say binary is a "base-2" numerical system. Decimal, which is our regular number system, has the digits 0-9. This means decimal is considered a "base-10" numerical system, since there are ten different digits we can use.

Binary is used in computers today because we can represent any binary string as a sequence of "on" and "off" switches. 1 represents on, and 0 represents off. Computers have billions of devices called transistors, which can be turned on and off. By switching on and off, these transistors are able to perform all the things modern computers are able to do.

As we continue with this lesson, we will learn how computers use binary to represent numbers and letters.
